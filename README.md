    x도의 기온일 때, 생산되는 쌀의 생산량(Total_Rice_Output 변수)</br>
    x는 12도 보다 높으며, 25도 보다 낮다는 가정을 한다.

    Output_Rice_function = 50*x*x + 0.3*x + 100</br>

    기온의 30프로와 100톤의 쌀을 합친 값만큼 증가한다.
